    Containers are the most basic layout element and are required when using the grid system.
    Containers are used to contain, pad, and (sometimes) center the content within them.
    While containers _can_ be nested, most layouts do not require a nested container.

    ## Basic container

    The basic container is a fixed width proportionate to screen size.

    <CWFCode code={example1} />

    ## Fluid container

    It is also possible to add a `.container--fluid` class to your container and it will have a max width of `100vw` regardless of screen size.
